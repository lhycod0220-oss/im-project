주제 1 : 고령화 시대 안전 지킴이 앱
1.주제 선정 이유
저출산 고령화 시대 스마트 기기 조작과 AI에 어려움을 겪는 어르신들을 위해, 복잡한 화면 대신 단순하고 어르신들도 친숙하게 접근 가능한 워치와 앱을 만드는게 목표입니다.

2.앱
홈 화면 : 메인 화면에 연동된 워치를 통해 현재의 심박수를 직관적으로 나타냅니다. ?버튼을 누를 시 챗봇이 나타나며 챗봇은 어르신들의 궁금증을 친절하게 알려줍니다.

몸 상태 : 병원에서 진단받은 질병이나 몸의 불편한 부분을 직관적으로 표시합니다. 
이를 관리하고 예방하는 방법을 동영상과 전문가의 지식을 뒷받침하여 어르신들께 상세한 정보를 제공합니다.

알약 : 메인화면에 있는 캘린더의 '오늘 날짜'를 누를 시 아침, 점심, 저녁으로 세 개의 체크박스가 나타납니다.
약을 복용하고 해당 시간에 체크박스를 체크하면, 체크박스를 누른 시점을 기준으로 복약 시간이 기록되어 하루 복약 여부를 쉽게 확인할 수 있습니다.
캘린더를 모두 채울 시 쿠폰 등 보상을 제공하여 동기를 부여합니다.

위치 : 연동된 워치를 통해 현재의 위치를 알려주고 워치에서 받은 사용자의 음성 데이터를 바탕으로 목적지를 설정해 효율적인 동선으로 길을 안내합니다.

Flutter(Dart) : 앱의 전체적인 UI 레이아웃 구현, 복약 캘린더 등의 내부 로직 데이터 처리 합니다.
Bluetooth Low Energy : 저전력 블루투스 통신으로 위치와 워치와 연동합니다.
LLM API : AI챗봇 기능을 구현합니다.
Firebase : 사용자의 데이터를 저장하고 실시간으로 동기화 합니다.
Python. Open API : 공공데이터포털의 Open API를 활용해 국가 검증 의료 정보를 가져와 학습시킵니다. 지도 API를 연동해 길 찾기를 구현합니다.

3.워치
심박수 : 심박수 센서를 통해 현재의 심장박동 수치를 실시간으로 알려줍니다.

시간 : 현재의 시간/요일/날짜를 알려줍니다.

위치 : 첫 화면에 나타나는 목저기 검색 버튼을 누르면 "목적지를 말씀해주세요"라는 음성(TTS)가 출력됩니다.
사용자가 말한 목적지를 연동된 스마트폰 앱에 현재의 위치와 목적지가 표시됩니다.
부가적으로 워치는 목적지를 화살표로 나타내어 직관적으로 길을 안내해줍니다.

C/C++(Arduino) : 아두이노를 활용해 심박수 센서, 마이크, 디스플레이 등 직접 제어하고 동작시킵니다.

Bluetooth Low Energy :  저전력 블루투스 통신으로 앱과 연동합니다.

주제 2 : 가상 기반 육아 시뮬레이션(모바일 앱)
1.주제 선정 이유
1인 가구 증가와 함께 반려동물을 키우는 사람들이 늘고 있습니다,
그만큼 충동적으로 입양한 사람들이 경제적부담, 시간/행동 등 현실적으로 다가오는 문제점으로 인해 유기견도 증가하고 있습니다.
위의 무책임한 입양을 방지하는게 목표입니다.

1.종류 : 말티즈, 푸들, 포메라니안, 골든 리트리버, 비숑 등 자신이 키울 강아지의 품종을 먼저 정합니다.

2.강아지의 현재 상태를 알 수 있는 UI를 보고(포만감, 스트레스 등)
유저는 사료, 물, 산책시키기 등 여러 상호작용을 통해 강아지의 문제점들을 해결합니다.

3.상호작용의 종류는 실제 견주분들의 겪으실말한 예민한 문제인 병원 치료비, 여행을 떠날때 펫시터에게 맡겨 발생하는 금액 등 현실적인 금액을 시뮬레이션에 집어 넣어 사람들에게 경각심을 줍니다.

Unity (언어: C#) : 2D 타일맵(방 내부) 환경과 상태창 UI를 구축하고, 강아지의 자율 행동 패턴(AI) 및 실시간 예산(통장 잔고) 차감 시스템을 구현합니다.
Godot Engine : 파이썬과 유사한 문법을 활용해 강아지의 내부 수치(포만감, 스트레스) 데이터를 관리하고, 돌발 청구서 이벤트 팝업 등 전체적인 게임 로직을 가볍고 빠르게 처리합니다.
